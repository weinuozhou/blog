# 前言

本书是一本对人类用户极其友善的大数据入门工具书, 从零开始学习大数据相关知识, 主要面向初学者

> 本书主要是在参考官方文档, 并结合作者学习使用经验基础上整理总结写成的

* 所有 SQL 语句例子, 是基于 `MySQL-5.7.19` 运行的
* 所有 Python 语句例子, 都是基于 `Python-3.9.7` 或 `Python-3.8.10` 运行的
